Hello there, Welcome to my portfolio.

I'm Tobi, a Data Analyst with years of experience in transforming data into visuals using SQL, Python and Tableau, amongst other data analytical tools. I am passionate about communicating and sharing data insights to stakeholders and decision makers to help achieve business goals. Below are some of my projects I have worked on.

WeRateDogs Twitter Analysis

Gathered, assessed, cleaned and analyzed 5,000 tweets from twitter user @dog_rates for quality and tidiness issues using tweepy and request library.

Product Order Analysis using MySQL

Queried an e-commerce data to answer business questions using MySQL and presentation of product order metrics on tableau dashboard.

Loan Data Exploratory Analysis

Performed a Univariate and Multivariate analysis of over 110,000 loan data to determine Loan status, amount, interest rate, income range, loan duration and relationship between it's variables.

Kenya School Data Analysis using R and Tidyverse

Analyzed a Kenya scholarship school programme dataset to answer specific questions using R programming language and Tidyverse packages.

FBI Gun Data Analysis

Investigated an FBI Gun dataset to determine overall trend of gun purchase by year, states, growth in gun registration, background checks and it's relationship using matplotlib and seaborn library.

Tableau Dashboards

Dashboards are useful for monitoring, measuring, and analyzing relevant data in key areas to stakeholders. These are my dashboards I have worked on ranging from sales data to e-commerce and movie data.
